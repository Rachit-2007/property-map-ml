# 🏠 Property Map ML

A full-stack Django project where users can:

## 🚀 Features
- 📍 Add properties on map
- 💰 Predict price using ML
- 📊 View statistics (Max, Min, Avg, Total)
- 🔍 Search & filter properties

## 🛠 Tech Stack
- Django
- HTML, CSS
- Leaflet.js
- Machine Learning (Scikit-learn)
